# Solving OpenAI Gym - MountainCar-v0

Solve MountainCar-v0 challenge with Q-learning table.

### Install

This project uses numpy and gym. Install depedencies with `pip`

	$ pip install -r requirements.txt

### Run

Run

	$ python train.py

Result

	$ python train.py
	----
	Solved in 1763 steps
